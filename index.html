Diabetes Logistic Regression

Python, Jupyter notebook, ScikitLearn, Pandas

I developed a logistic regression model to predict diabetes risk using the Pima Indians Diabetes Database. After cleaning the data and establishing a baseline accuracy, I optimized the model through regularization and threshold adjustments. The best-performing model achieved 72.3% recall and 64.2% precision, balancing sensitivity and specificity. I further tested the model using a Stochastic Gradient Descent (SGD) classifier, achieving a strong balanced accuracy of 86.64%. This project showcases my skills in data cleaning, model optimization, and handling class imbalance.

CLV Prediction

I built a model to predict Customer Lifetime Value (CLV) for an auto insurance company, focusing on data cleaning, feature engineering, and model optimization.
